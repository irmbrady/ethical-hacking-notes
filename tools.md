# Hacking tools

## Network security monitoring tools

- [Argus](https://sectools.org/tool/argus/) - Real Time Flow Monitor designed to track and report on the status and performance of all network transactions seen in a data network traffic stream.
- [p0f](https://tools.kali.org/information-gathering/p0f) - Passive traffic fingerprinting mechanisms to identify the players behind any incidental TCP/IP communications (often as little as a single normal SYN) without interfering in any way.
- [Nagios](https://www.nagios.org/) - Infrastructure monitoring
- [OSSEC](https://www.ossec.net/) - Host-based Intrusion Detection System
- [Splunk](https://www.splunk.com/) - A software platform to search, analyse and visualize the machine-generated data gathered from the websites, applications, sensors, devices etc. which make up your IT infrastructure and business.
